# .NET8 Web API Study(ASP.NET Core REST API)

ASP.NET Core Web API 학습을 위해 진행한 프로젝트입니다.  
JWT 인증과 사용자별 포트폴리오 조회 기능을 구현하며  
API 구조와 인증 흐름을 이해하는 데 중점을 두었습니다.

## Tech Stack
- .NET 8 / ASP.NET Core
- Entity Framework Core
- SQL Server
- JWT Authentication
- Swagger

## Features
- JWT 기반 인증 API (로그인, 토큰 검증)
- 사용자별 포트폴리오 조회 API
- 사용자 정보 조회 API

## Key Learnings
- JWT 인증 흐름 (토큰 발급 → 검증 → 인증 처리) 이해
- ASP.NET Core에서 인증 미들웨어 및 요청 흐름 이해
- EF Core를 활용한 데이터베이스 접근 및 관계 설정 경험
- Controller 기반 API 설계 및 계층 구조 이해
